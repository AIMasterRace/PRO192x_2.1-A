PRO192x_2.1-A notes
1. Cấu trúc của 1 file Java:
    - Tên class, phải trùng với tên file
        + VD: public class Main {} (với file Main.java)
    - Tên hàm main(), bắt buộc cần phải có trong mỗi chương trình
        + VD: public static void main(String[] args) {} (cố định)
    - helloWorld:
        'public class helloWorld { d
            public static void main(String[] args) {
                System.out.println("Hello World!");
            }
        }'

2. Chú thích
    - Nếu chú thích 1 dòng thì cần // đầu dòng hoặc sau lệnh
    - Nếu chú thích nhiều dòng thì thêm chú thích trong cặp dấu /**/

3. Biến & Kiểu DL
    - Các biến được khai báo theo định dạng: type variable = value;
        + VD: int a = 10;
    - Các biến thường dùng: int, float, char, boolean, double, String (lưu ý chữ S ghi hoa)
    - Ép kiểu DL: dùng type variable = (castType) value;
        + VD: double a = 9.78d
              int b = (int) a;
